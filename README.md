# yourstrategy-site

The root landing page for **Your Strategy**, served at
[yourstrategy.co](https://yourstrategy.co) via GitHub Pages.

A small personal hub: a tagline and links out to each app.

- `index.html` — the hub
- `styles.css` — stylesheet
- `assets/` — app icons for the cards
- `CNAME` — apex custom domain for GitHub Pages
- `.nojekyll` — serve files as-is (no Jekyll build)

Apps linked:
- Reminder Wrangler → https://reminderwrangler.yourstrategy.co
- Reverdure → https://reverdure.yourstrategy.co

Plain static HTML/CSS, no build step. Edit and push to `main`; GitHub Pages redeploys.
